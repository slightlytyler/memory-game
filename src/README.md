Simple Template
===============

A simple scaffolding template to initialize projects.

- Based on the html5 boilerplate.
- Set up with gruntJS: Includes gruntfile.js and package.json
- Included grunt tasks:
	- sass
	- uglify
	- autoprefixer
	- watch
	- clean
	- concat
	- modernizr
	- imagemin
	- htmlmin
	- usemin
